# Assignment1_Zhanerke
Heterogeneous Parallelization


Zhanerke Duisen, ADA-2403M

Задание по C++ и OpenMP
__________________________________________________________________________________________________________________________
Описание задание

В этом assignment1 реализованы 4 задания на языке C++ с использованием динамических массивов, генерации случайных чисел и параллельных вычислений с помощью OpenMP. Основная цель — практическая работа с динамической памятью, измерение времени выполнения алгоритмов и изучение ускорения вычислений при использовании многопоточности.

Блок-схемы в папке "БЛОК-СХЕМЫ". (Задание 2 и Задание 3 объедененная)
__________________________________________________________________________________________________________________________
ЗАДАНИЕ 1 (assignment1_task1(Zhanerke).cpp)

Создание динамического массива из 50 000 целых чисел.

Заполнение массива случайными числами от 1 до 100.

Вычисление среднего значения элементов массива.

Освобождение памяти, выделенной под массив.

Решение задание 2 и задание 3 в одном файле cpp.

__________________________________________________________________________________________________________________________
ЗАДАНИЕ 2 (assignment1_task2(Zhanerke).cpp)

Создание массива из 1 000 000 целых чисел.

Последовательный поиск минимального и максимального элементов.

Замер времени выполнения поиска.

___________________________________________________________________________________________________________________________
ЗАДАНИЕ 3 (assignment1_task2(Zhanerke).cpp)

Параллельный поиск минимального и максимального элементов массива (1 000 000 чисел).

Замер времени выполнения алгоритма.

Сравнение времени с последовательной версией.

___________________________________________________________________________________________________________________________
ЗАДАНИЕ 4 (assignment1_task4(Zhanerke).cpp)

Создание динамического массива из 5 000 000 целых чисел.

Вычисление среднего значения последовательным способом и с использованием OpenMP с редукцией.

Замер времени выполнения обоих алгоритмов.

Сравнение эффективности параллельной версии.
